# ide-sync
# 🤖 ابزار مدیریت پروژه با هوش مصنوعی (بدون نیاز به API)

این ابزار به شما امکان می‌دهد از **هر هوش مصنوعی** (ChatGPT, Claude, Gemini و...) برای مدیریت پروژه‌های خود استفاده کنید، **بدون نیاز به API key**!

## 💡 چگونه کار می‌کند؟

1. **شما**: پروژه را به برنامه می‌دهید
2. **برنامه**: پروژه را به JSON تبدیل می‌کند
3. **شما**: JSON را به هر چت AI می‌برید
4. **AI**: تغییرات را در JSON اعمال می‌کند
5. **شما**: JSON جدید را به برنامه می‌دهید
6. **برنامه**: تغییرات را روی پروژه اعمال می‌کند

## ✨ ویژگی‌ها

- 🆓 **رایگان**: نیازی به API key نیست
- 🔒 **امن**: پروژه روی سیستم شما می‌ماند
- 🤖 **چندمنظوره**: با هر AI کار می‌کند
- 🔀 **یکپارچه با Git**: مدیریت خودکار branch و commit
- 📊 **نمایش تغییرات**: diff قبل از اعمال
- ✅ **کنترل کامل**: تایید/رد تغییرات
